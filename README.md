# JamesTech — marketing site

Static site for **jamestech.com** (Mark Lucas / JamesTech LLC). No build step; files
are served from the repo root by Netlify.

This is the JamesTech-branded sibling of the Velocity AI audit site. Same copy, CTAs,
and offer ladder; rebranded to the JamesTech name, the gold JT^AI mark, and the
bright blue + gold palette.

## Pages

- `index.html` — 90-Minute AI Audit landing page (funnel front end)
- `case-study.html` — anonymized financial-services coaching case study (shareable link)
- `privacy.html`, `terms.html`, `eula.html`
- `qbo-callback.html` — QuickBooks OAuth callback

## Brand

- Colors: royal blue `#2A66D6`, deep blue `#1A48A6`, navy `#0E2C6E`, gold `#FFC836`→`#FFB200`, white. Gold is an accent only.
- Fonts: Space Grotesk (display) + Manrope (body), Instrument Serif (italic accents).
- Logo: gold-accented JT^AI badge, navy wordmark. Assets in `/assets`, `favicon.ico` at root.

## Deploy

Static Netlify deploy from `main`. Booking CTA points to the GoHighLevel calendar.
